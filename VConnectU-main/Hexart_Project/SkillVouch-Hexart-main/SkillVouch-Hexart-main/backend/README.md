# SkillVouch Backend

## Setup
```bash
npm install
```

## Environment Variables
Create `.env` file:
```
MONGODB_URI=mongodb+srv://USERNAME:PASSWORD@cluster0.nkgnqd5.mongodb.net/skillvouch?retryWrites=true&w=majority
MISTRAL_API_KEY=your-mistral-api-key
```

## Development
```bash
npm run dev
```

## Deployment
Deploy to Vercel with this folder as the root directory.
